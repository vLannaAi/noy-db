## Summary

- **#83 plaintextTranslator hook**: Consumer-supplied async translator for `i18nText` fields with `autoTranslate: true`. Runs before i18n validation in `put()`, caches by content hash, cleared on `db.close()`. `db.translatorAuditLog()` returns audit entries with cache-hit flag. `TranslatorNotConfiguredError` when translator is missing but required.
- **#84 exportStream dictionary snapshot**: `exportStream()` captures all dict snapshots atomically before the first yield — concurrent mutations don't affect the export. Attaches `dictionaries` field to chunks with dictKey collections. `exportJSON()` embeds `_dictionaries`; omits it when `resolveLabels` is set.
- **#85 Query DSL dictKey integration**: `query().join(field)` resolves dictKey fields as dict joins, attaching `{ key, ...labels }` under the alias. `groupBy(field).aggregate(...).runAsync({ locale })` adds `<field>Label` to each bucket row. Grouping always uses stable keys regardless of locale. `DictionaryHandle` write-through sync cache powers O(1) join snapshots.
