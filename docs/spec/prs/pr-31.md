## What ships in v0.3.0

**New packages:**

- \`@noy-db/nuxt\` — Nuxt 4 module with auto-imports, SSR-safe runtime, devtools tab, typed config

**Power features (all in \`@noy-db/core\`):**
- Reactive query DSL with chainable builder, 10 operators, \`.live()\` refs
- Secondary indexes — 4–6× speedup vs linear scan on 10K records
- Pagination via \`listPage\` adapter capability + streaming \`scan()\` async iterator
- Lazy hydration + LRU eviction for collections that don't fit eager mode

**Adapters:** memory, file, browser, dynamo, s3 all implement \`listPage\`.

**Reference demo:** \`playground/nuxt/\` — Nuxt 4 + Pinia accounting demo, the integration test for the entire adoption story.

**Docs:** complete sweep of getting-started, end-user-features, architecture, adapters, deployment-profiles, README. Bug fix: 80+ stale \`@noydb/*\` package scope references corrected to \`@noy-db/*\`.
