Advanced cryptographic primitives beyond the AES-256-GCM + PBKDF2 core. Every primitive here is a distinct feature flag, never a default. Candidates: deterministic encryption mode (searchable encrypted indexes; opt-in because it leaks equality), Shamir secret-sharing for KEK (k-of-n multi-party unlock), zero-knowledge proofs for compliance (prove properties without revealing values), homomorphic aggregates on a sub-collection.
